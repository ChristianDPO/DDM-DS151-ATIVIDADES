# Ferramentas e Versões

- [Node 16.15.1](https://nodejs.org/en/), Recomendado instalar via [nvm](https://github.com/nvm-sh/nvm)
```
    sudo apt-get install nvm
    nvm install 16.15.1
    nvm use 16.15.1
    node -v
```
- Pacotes do Node
```
npm install -g expo-cli
```
